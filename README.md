# Subscription Tracker

A full-stack subscription management system with user authentication and CRUD functionality.

## Features
- User login & register
- Add/Edit/Delete subscriptions (coming soon)
- Pause subscriptions (coming soon)
- Monthly spending calculation (coming soon)

## Tech Stack
- Frontend: HTML, CSS, JavaScript 
- Backend: Node.js, Express
- Database: MongoDB

## How to run
-Clone the repository.
-Install dependencies:
```bash
npm install
```
-Start the application 
```bash
 npm start
```
